Hướng dẫn kết nối
1

Mở Google Sheet

Vào sheets.google.com → mở Sheet đã tạo sẵn → menu Extensions → Apps Script.

2

Paste code script

Xóa code cũ → paste toàn bộ nội dung file Code.gs → nhấn Save → chọn setupSheets → nhấn Run.

3

Deploy Web App

Nhấn Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy → copy URL.

4

Dán URL vào Cài đặt

Paste URL vào ô Script URL ở tab Cài đặt → nhấn Lưu & kết nối → xong!
